# ida-bridge

Atomic plugin exposing only `ida_bridge`.

## Tool

- `ida_bridge`

## Environment

- `IDA_BRIDGE_URL` (default: `http://127.0.0.1:18081`)
- `PLUGIN_IDA_BRIDGE_ENABLED` / `PLUGINS_IDA_BRIDGE_ENABLED`

## Notes

- Endpoint is restricted to loopback hosts (`127.0.0.1` / `localhost` / `::1`).
